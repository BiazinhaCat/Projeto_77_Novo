Sobre VSCode

O que é VS Code?

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Como configurar o VS Code?

1 - Baixe e instale a versão mais recente do Visual Studio Code para seu sistema operacional. Se você já tem o Visual Studio Code instalado, não é necessário reinstalá-lo. 2 - Inicie o Visual Studio Code. 3 - Na barra de ferramentas à esquerda, clique no ícone de extensões. 4 - Pesquise o Salesforce Extension Pack (Pacote de extensões do Salesforce) e clique em Install (Instalar).

E por que o VS Code é ultilizado?

Por que são para projetos individuais e salvas na pasta do projeto. Por fim, essas são as razões pelas quais o VSCode é um editor de código incrível que você não deve ignorar.

E como abrir a tela do console?

Para abrir a tela do console aperte a tecla F12 no seu computador.

E como imprimir qualquer coisa no console?

Apenas abra o console, com Ctrl+Shift+K ou F12 , e, na parte superior direita, você verá um botão que diz "Switch to multi-line editor mode" (Mudar para o modo editor multilinhas). Como alternativa, pressione Ctrl+B .
